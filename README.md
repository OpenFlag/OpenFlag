# Introduction

OpenFlag is an open source feature flagging, A/B testing and dynamic configuration service.

# Documentation

You can find documentation in <a href="https://openflag.github.io">here</a>.

### Supported Rules

* `contains`
* `excludes`
* `intersection`
* `match`
* `random`
* `rollout`
* `union`
* `<`
* `<=`
* `>`
* `>=`
* `!`
* `%`
